Welcome to the help page

How to create cards:

To begin, simply click the "create" link at the top of the page. Once you are on the create page, click on the dropbox near the top of he page to select a holiday.CLick the browse button next and a window will appear to allow the user to select an image from their compoter. Once the holiday and user photo has been selected has been selected, click the submit button. Your cards will be created automatically. Download buttons will apear below the cards to save the images.

Why are images distored?

User images are automatically resized to fit the card templates. Using larger images is recommended. Large image sizes can be reduced while still retaining the quality, however the same is not true for smaller images. For best results, we recommend using an image size of at least 950x530 or larger where the width is greater than the height.

How to save cards:

After the cards have been created, there will be a download button under each card. CLick the download button to automatically save the image to your device. A second option would be to right click on the picture and select save image.
